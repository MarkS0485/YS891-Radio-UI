# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest `main` | :white_check_mark: |

## Reporting a Vulnerability

Please report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/MarkS0485/YS891-Radio-UI/security/advisories/new)
rather than opening a public issue. You can expect an initial response within a
week. Accepted reports are fixed on `main` and credited in the advisory unless
you prefer otherwise.
